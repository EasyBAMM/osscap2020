## install
카메라를 연결 후 라즈베리파이 킬 것.  
```
1. git clone https://github.com/EasyBAMM/osscap2020.git
2. cd osscap2020
3. sudo pip3 install flask
4. sudo pip3 install flask_cors
5. python3 main.py
```
web browser url : http://localhost:5000 접속
